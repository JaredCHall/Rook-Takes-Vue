# Rook Takes

Personal Project about the game of chess.

### Features

- Game Analysis Board
- Puzzles
- Visualization Exercises

## Install and Start
- Clone the repo
- `npm install`
- `npm start`

# Stack
- SPA on top of riot.js webpack loader
- Sass css preprocessing
- No idea how to do routing yet, but there's a riot module for it
